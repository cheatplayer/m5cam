# M5Cam

**M5Stack Camera with PIR**

## Dev

```
export PATH=$HOME/workspace/hardware/xtensa-esp32-elf/bin:$PATH
export IDF_PATH=~/workspace/hardware/esp-idf
```

```
$ make
$ make flash

$ make monitor
```

## Config

```
$ make menuconfig
```
